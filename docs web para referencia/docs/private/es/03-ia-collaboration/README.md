# 🤖 Documentación de Colaboración con IAs - Español

Guías y procedimientos para modelos de IA colaborando en ControlApp.

## 📁 Contenidos

- **ONBOARDING_FOR_NEW_AIs.md** - Guía rápida de onboarding (copy-paste en chat)
- **HOW_TO_SWITCH_TO_NEW_AI.md** - Cómo cambiar a una nueva IA entre sesiones
- **LOCALE_RESOLVER_SYSTEM.md** - Explicación del sistema resolvedor de locale

## 🎯 Propósito

Documentación específica para IAs:
- Cómo trabajar en el proyecto
- Normas y convenciones esperadas
- Flujos de decisión claros
- Procedimientos exactos

## 👉 Si eres una IA

**Primera vez**:
1. Copia completamente: **ONBOARDING_FOR_NEW_AIs.md**
2. Pega en el chat
3. Espera confirmación
4. Luego lees: **AI_GUIDELINES.md**

**Próxima sesión con nueva IA**:
→ Lee: **HOW_TO_SWITCH_TO_NEW_AI.md**

## 🤖 Instrucción Principal para IAs

```
❌ NO: Crear documentos nuevos "SESSION_SUMMARY_*.md"
✅ SÍ: Actualizar CHANGELOG_DETAILED.md
```

Ver AI_GUIDELINES.md sección "Patrón de Documentación" para detalles.
